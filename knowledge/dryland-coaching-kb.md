## 1. Why dryland for sprint freestyle

- Dryland builds the **force, power and stability** that the pool can't load directly; for sprinters specifically it develops the pulling force and trunk stability that translate to higher distance-per-stroke under fatigue [source: https://www.yourswimlog.com/sprint-freestyle-dryland/] [source: https://www.nsca.com/education/articles/nsca-coach/beyond-the-pool-improving-swimming-performance-with-dryland-training/].
- Maximal-strength and power work, not endurance circuits, drives sprint performance — the 50 is a force/power event.
- Core strength matters because **water exploits any weakness in body position**; a stable trunk lets force transfer from the catch through the hips.

---

## 2. The stability → strength → power continuum

Build in order; don't skip stages [source: https://www.yourswimlog.com/sprint-freestyle-dryland/]:

1. **Stability** — joint control and coordination (hips, shoulders, core). Establishes safe positions to load.
2. **Strength** — maximal force production (heavier loads / harder bodyweight progressions).
3. **Power** — rate of force development (explosive/plyometric work) built *on top of* strength.

**Periodization** (typical swim blocks) [source: https://www.swimmingscience.net/dryland-periodiziation-swimmers/] [source: https://swimswam.com/simple-effective-dryland-periodization-planning-for-fast-swimming/]:
- Core stabilization (~4 weeks) → muscular power (~4 weeks) → power endurance (~4 weeks).
- Maps onto the pool phases: Phase 1 dryland = stability + base pulling strength + core; Phase 2 = power (incl. plyos once cleared); Phase 3 = light activation/maintenance.

**⚠️ Julian's current stage:** stability + controlled strength. Plyometrics are **deferred** until the quad/hip-flexor base is established and landing mechanics are taught — see §8.

## 3. Session structure

A dryland session has three parts [source: https://swimswam.com/how-to-create-the-best-dryland-workouts-for-high-school-swimmers/]:
1. **Activation / warm-up** — functional movements, CNS activation, swim-specific movement patterns.
2. **Dynamic / power** — mobility + athleticism + (later) power work.
3. **Strength** — the main loading block.

Julian's working template (from his logs), ordered for swim-specificity:
- **Block A — Core & rotation** (always first; core is his primary weakness)
- **Block B — Pulling strength** (lat/rhomboid dominant)
- **Block C — Shoulder stability & injury prevention**
- **Block D — Controlled quad/hip-flexor loading** (no ballistic work yet)

Note: doing inverted rows *before* pull-ups is the more swim-specific order but pre-fatigues pull-up totals — keep the order consistent for valid comparisons.

## 4. Pulling strength (lat-dominant — the sprint engine)

Pull-ups are the gold-standard swimmer exercise — they mimic the freestyle pull and load lats, delts, biceps and grip [source: https://swimlikeafish.org/6-dryland-exercises-for-a-stronger-freestyle-pull].

| Exercise | Swim rationale | Equipment |
|---|---|---|
| Pull-ups | Direct freestyle-pull pattern; lat power | bar / rings |
| Ring rows / Inverted rows | Scapular retraction, horizontal pull, catch strength | rings / low bar |
| Lat pulldown (band) | Isolated pull power/endurance, symmetry | band |
| Straight-arm pulldown (band) | EVF/catch-specific lat loading | band |
| Dips | Pressing balance to the pull-heavy program | bars / rings |
| ~~Dumbbell lat pullover~~ | **RETIRED — caused shoulder discomfort** | — |

## 5. Core & rotation (freestyle-specific)

Train **anti-rotation** (resisting twist) and **rotation** (generating it), plus trunk stability [source: https://pmc.ncbi.nlm.nih.gov/articles/PMC8214624/].

| Exercise | Targets | Equipment |
|---|---|---|
| Hollow-body hold | Total-trunk tension, streamline shape | none |
| Plank / plank-to-push-up | Anti-extension stability | none |
| Bird dog | Contralateral stability, anti-rotation | none |
| V-ups | Dynamic trunk flexion | none |
| Hanging knee raises | Lower-ab + hip-flexor control | bar |
| L-sit (progression) | Compression + hip-flexor strength | bars/rings ⚠️ quad |
| Russian twist / chop | Rotational power | band/DB |
| Med-ball slam | Explosive trunk (power phase) | med ball |

## 6. Shoulder stability & injury prevention

A pull-heavy program needs deliberate shoulder care to stay healthy [source: https://www.usms.org/fitness-and-training/articles-and-videos/articles/dryland-exercises-to-improve-your-freestyle].

| Exercise | Targets | Equipment |
|---|---|---|
| Prone Y-T-W raises | Lower-trap / scapular control | none / light DB |
| Band external rotation | Rotator-cuff health | band |
| Scapular pull-ups | Scapular control under load | bar |
| Support hold (dip/ring) | Shoulder stability, scapular depression | bars/rings |
| Bar-hang external rotation | Overhead mobility + cuff (replaces lat pullover) | bar |

## 7. Controlled leg / quad–hip-flexor loading (Julian priority)

Build the base **before** any explosive push-off / plyometric work [source: https://www.yourswimlog.com/prevent-muscle-cramps-swimming/]. Glutes/hamstrings/hip-flexors support propulsion and prevent fatigue-based cramps.

| Exercise | Targets | Equipment |
|---|---|---|
| Bar-assisted deep-squat hold | Hip/quad mobility + isometric base | bar |
| Bar-assisted single-leg squat | Unilateral quad strength, control | bar |
| Slow standing hip-flexor drive | Hip-flexor endurance (his deficit) | none |
| Glute bridge / single-leg | Posterior chain, anti-cramp | none |
| Single-leg deadlift | Hamstring/glute, balance | none / DB |
| Wall-sit | Quad isometric base | none |

**Rule for active quad flag:** controlled tempo only, no ballistic/plyometric work, keep isometric holds (L-sit, deep squat) short and progress slowly; stop on any tightness.

## 8. Plyometrics & explosive power (when to introduce)

Plyometrics develop leg/pelvis explosiveness for the start and push-off; 8 weeks of plyo work has improved 15/25/50m times by up to 1.5s [source: https://www.yourswimlog.com/benefits-of-plyometric-training-for-swimmers/].

Key principles:
- **Horizontal, not vertical.** Swim starts project the body horizontally — favour **broad jumps** over jump-height work [source: https://swimswam.com/plyometrics-for-swimming-the-best-dryland-jumps-that-are-swim-specific/].
- **Teach landing first.** Use depth drops to teach force *control* before creating more force [source: https://www.traindaly.com/train-daly/blog/plyometric-exercises-for-swimmers].
- **Low volume.** ≤20 foot contacts/session; the goal is power, not conditioning. Start slowly.

**⚠️ Julian:** plyometrics are **deferred**. Gate them behind: (a) quad/hip-flexor base established across full sets without pre-cramp, and (b) landing mechanics taught via controlled depth drops. Re-evaluate once the §7 base is stable.

## 10. Loading / sets-reps guidelines

| Goal | Load | Reps | Notes |
|---|---|---|---|
| Max strength / power | 70–85% 1RM (or hard bodyweight) | **3–6** | Sprint priority [source: https://www.nsca.com/education/articles/nsca-coach/beyond-the-pool-improving-swimming-performance-with-dryland-training/] |
| Muscular endurance | 50–70% 1RM | 10–15 | Secondary for a sprinter |
| Isometric holds | bodyweight | 10–30s | Progress duration slowly (esp. hip-flexor for Julian) |
| Plyometrics (when cleared) | bodyweight | ≤20 contacts | Power, not conditioning |

Volume: 45–60 min total; list estimated time per block.

## 11. Tracking & progression markers (deterministic)

Dryland has **no Garmin CSV** — input is a structured form. Progression markers the engine tracks in `dryland_baselines`:

| Marker | Definition | Progress = |
|---|---|---|
| Rep PR | best set / total reps for an exercise | more reps at equal/harder progression |
| Hold-duration PR | longest isometric hold (plank, L-sit, support) | longer hold, clean form |
| Load PR | weight used (when DBs available) | more load at equal reps |
| Bilateral symmetry | L vs R single-leg/single-arm output | gap narrowing |
| Cross-feed flags | injury events (e.g. L-sit quad pre-cramp) | **constrain next pool + dryland** |

Confounders to record (so comparisons stay valid): grip conditions (slippery bars), exercise order (pre-fatigue), and overall fatigue (day after a hard pool session).
